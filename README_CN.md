# 💧 腹膜透析患者健康管理小程序

[🇺🇸 English Version](README.md)

---

## 🧭 概览

本项目旨在为 **腹膜透析患者** 提供便捷的健康数据管理与透析记录支持。  
通过微信小程序实现数据记录、提醒设置与趋势可视化功能。

> ⚠️ **注意：** 本项目仍在开发中，部分功能尚未完成。

- **前端仓库：** [touxi-riji-web](https://github.com/greysonOuyang/touxi-riji-web) — 基于 **Taro + React + TypeScript + NutUI**
- **后端仓库：** [futou-riji-backend](https://github.com/greysonOuyang/futou-riji-backend) — 基于 **Spring Boot**

---

## ⚙️ 核心功能

| 模块 | 功能说明 |
|------|-----------|
| **首页仪表盘** | 显示每日透析概览、关键健康指标与快捷入口 |
| **健康记录** | 支持记录腹透、体重、血压、心率、血糖等多项健康数据 |
| **提醒设置** | 提供透析、用药、就诊等提醒通知 |
| **统计与趋势** | 展示历史记录与关键健康趋势 |
| **个人中心** | 用户资料管理、APP 设置及个性化选项 |

---

## 🧠 技术架构

| 层级 | 技术栈 | 说明 |
|------|----------|------|
| **前端** | Taro + React + TypeScript | 构建微信小程序界面 |
| **后端** | Spring Boot + MyBatis + MySQL | 提供数据存储与健康记录管理 |
